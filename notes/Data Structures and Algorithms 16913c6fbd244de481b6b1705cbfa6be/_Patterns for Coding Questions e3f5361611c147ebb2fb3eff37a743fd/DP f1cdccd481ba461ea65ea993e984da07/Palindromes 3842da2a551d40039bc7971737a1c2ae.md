# Palindromes

[Coding Patterns: Palindromes (DP)](https://emre.me/coding-patterns/palindromes/)

```python
# 0(n) time | O(1) space - where n = len(string)
def isPalindrome(string):

    left = 0
    right = len(string) - 1
    while left <= right:
        if string[left] != string[right]:
            return False

        left += 1
        right -= 1

    return True
```
